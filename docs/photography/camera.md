# 相机使用

本文介绍摄影最核心的三个曝光参数，以及常用拍摄模式与对焦方式。掌握这些基础后，绝大多数场景都能拍出正确曝光的照片。

---

## 曝光三角

曝光量由三个参数共同决定，改变其中任意一个都会影响画面的亮暗，同时也会带来其他效果上的变化。

### ISO（感光度）

ISO 表示感光元件对光线的敏感程度。

- **数值越低**（ISO 100）：噪点少，画质干净，适合光线充足的环境
- **数值越高**（ISO 3200+）：进光能力强，但会产生明显颗粒感（噪点）

::: tip 经验法则
尽量使用相机的**原生最低 ISO**（通常是 ISO 100 或 64），只有在快门/光圈无法满足需求时才提高 ISO。有双原生 ISO 的机型，高感时优先跳到高原生点。
:::

::: details 噪点的形成原理

数字噪点本质上是信号放大过程中引入的误差，主要来自两个来源：

**光子散粒噪声（Shot Noise）**
光本身是量子化的。即使在完全相同的曝光条件下，每次打到感光元件某个像素上的光子数量也会随机波动。进光量越少（暗部、低曝光），这种波动相对于信号本身的比例越大，表现为画面暗部的颗粒感。这类噪点是物理规律决定的，无法消除，只能通过增加进光量（更大光圈、更慢快门）来改善。

**读出噪声（Read Noise）与放大噪声**
感光元件将光信号转换为电信号后，需要经过模拟放大电路才能被 ADC（模数转换器）读取。提高 ISO 本质上是提高这个放大倍数——信号被放大的同时，电路自身产生的电气噪声也一并被放大，形成随机分布的彩色噪点（色噪）和亮度噪点（亮噪）。

:::

::: details 各 ISO 典型噪点表现
| ISO | 噪点等级 | 适用场景 |
|-----|----------|----------|
| 100–400 | 极少 | 室外晴天、人像补光 |
| 800–1600 | 轻微 | 室内、阴天 |
| 3200–6400 | 明显 | 夜景、演出现场 |
| 12800+ | 严重 | 极端低光，慎用 |
:::

::: details 两种噪点的外观区别
- **亮度噪点**：像黑白胶片的颗粒感，明暗随机分布，相对容易接受
- **色彩噪点**：像素点呈现红绿蓝随机色斑，视觉上更明显，后期降噪时优先处理
:::

::: details 双原生 ISO（进阶）

部分相机（尤其是索尼、松下的视频导向机型，如 Sony FX3、Panasonic S5II）的感光元件内置了**两套独立的模拟放大电路**，分别针对低感光和高感光场景做了优化。相机会在两个特定的 ISO 档位之间切换电路，这两个档位称为**双原生 ISO**。

在双原生 ISO 的两个原生点上，放大倍数最低、读出噪声最小，信噪比最优，画质显著优于相邻 ISO 值。

| 机型 | 低原生 ISO | 高原生 ISO |
|------|-----------|-----------|
| Sony FX3 / A7S III | ISO 80 | ISO 12800 |
| Panasonic S5 II | ISO 100 | ISO 640（视频） |
| BMPCC 6K | ISO 400 | ISO 3200 |

实际拍摄时，优先使用这两个档位；需要介于两者之间的感光度时，**靠近低原生 ISO 往上调**比从高原生 ISO 往下调噪点更少。

> 注意：部分机型的双原生 ISO 仅在特定录制格式（如 S-Log / RAW 视频模式）下才完全发挥作用，照片模式下可能与普通 ISO 无差异，请查阅具体机型文档。

:::

::: details EI（电影机曝光指数）

电影摄影机（如 ARRI ALEXA、RED、Sony VENICE）通常不用 ISO，而是用 **EI（Exposure Index，曝光指数）** 来标定感光度。

两者的本质区别在于调整方式：

- **ISO** 改变的是传感器或放大电路的实际增益，直接影响原始信号
- **EI** 不改变传感器的实际增益，只是告诉相机/监视器**如何解读和显示这个信号**。传感器始终以固定的原生 ISO 进行拍摄，EI 的偏移仅作用于 LUT 或波形监视器的参考值，用于辅助现场曝光判断

这样设计的好处是：无论 EI 设置为多少，传感器的物理噪点特性始终保持在最优状态。调整 EI 本质上是在**后期空间内预先选择用哪段动态范围**，而不是真正改变硬件增益。

| 调整方式 | 实际增益变化 | 噪点变化 | 动态范围分配 |
|----------|-------------|----------|-------------|
| 提高 ISO  | 是 | 增加 | 固定 |
| 提高 EI   | 否 | 不变 | 向暗部倾斜（高光保护更多） |
| 降低 EI   | 否 | 不变 | 向亮部倾斜（暗部保护更多） |

> EI 的概念主要出现在专业电影制作流程中，使用 Log 格式拍摄时才有实际意义。对于日常摄影或使用普通视频模式的相机，仍以 ISO 为准。

:::

### 快门速度

快门速度决定感光元件暴露在光线下的时长，用秒或分数表示（如 1/500s、2s）。

- **高速快门**（1/500s 以上）：凝固运动瞬间，画面清晰锐利
- **低速快门**（1/30s 以下）：运动主体会产生模糊轨迹，有时是创作效果（水流、车流）

::: warning 手持拍摄的安全快门
手持相机时，快门速度低于 **1/焦距** 容易产生手抖虚化。
例如使用 50mm 镜头，安全快门约为 **1/50s** 或更快。
开启机身/镜头防抖（IBIS/OIS）可获得 3–5 档补偿。
:::

::: details 长曝光

快门速度低于约 **1s** 时进入长曝光范畴，需要三脚架固定相机。长曝光的核心是利用时间积累来呈现运动轨迹：

| 场景 | 典型快门 | 效果 |
|------|---------|------|
| 流水/瀑布丝滑感 | 1/4s – 2s | 水流变为柔顺白丝 |
| 城市车轨 | 10s – 30s | 车灯拉成光线条纹 |
| 星轨（短弧） | 15s – 30s | 星点保持点状，略有拖尾 |
| 星轨（完整圆弧） | 数小时连拍后期叠加 | 以北极星为圆心的同心弧 |
| 烟花 | 2s – 6s | 完整绽放轨迹 |

**操作要点**
- 使用**快门线**或相机内置**定时器（2s 延迟）**，避免按快门时产生震动
- 开启**反光镜预升**（DSLR）或电子前帘快门，进一步减少机械震动
- 对焦完成后切换到 **MF**，防止长曝过程中 AF 重新搜索
- 夜间长曝注意**传感器发热**，连续多张后可适当间隔散热

:::

::: details ND 减光镜

ND（Neutral Density，中性灰度）滤镜是均匀减少进光量的光学附件，不改变色彩，只降低亮度——相当于给镜头戴上墨镜。在光线充足的环境下，想要使用大光圈或低速快门时，即使 ISO 已压到最低，曝光仍会过度。ND 滤镜让你在白天也能实现浅景深（大光圈）或运动模糊（低速快门）。

**ND 档数与透光率**

ND 滤镜的规格通常以「ND 数字」或「档数（stops）」标注，每增加 1 档相当于快门速度减慢一倍：

| 标注 | 档数 | 透光率 | 快门补偿示例（原 1/500s） |
|------|------|--------|--------------------------|
| ND2  | 1 档 | 50%    | → 1/250s                 |
| ND4  | 2 档 | 25%    | → 1/125s                 |
| ND8  | 3 档 | 12.5%  | → 1/60s                  |
| ND64 | 6 档 | 1.6%   | → 1/8s                   |
| ND1000 | 10 档 | 0.1% | → 2s                    |

**固定 ND vs 可变 ND**

- **固定 ND**（如 B+W、Hoya）：光学质量稳定，无色偏，但需要多片覆盖不同档位
- **可变 ND**（两片偏振镜叠加旋转）：一片覆盖约 2–8 档，出行方便；旋转至极限时会出现「X 形暗角」，中低价位产品易有偏色

视频拍摄时快门速度通常固定为**帧率的两倍分之一**（如 25fps → 1/50s），晴天光线必然过曝，因此可变 ND 是视频拍摄的常备配件。

:::

### 光圈（F 值）

光圈控制镜头的进光孔径，以 f 数表示。f 数越小，光圈越大。

- **大光圈**（f/1.4、f/2）：进光多，景深浅，背景虚化明显
- **小光圈**（f/8、f/11）：进光少，景深深，全图清晰，适合风景

<ApertureDemo />

::: tip 最佳成像光圈
大多数镜头在 **f/5.6–f/8** 时分辨率和衍射的平衡最优，画质最佳。
大光圈（全开）和极小光圈（f/16 以上）通常画质有所下降。
:::

### 三者联动：曝光值（EV）

一张照片的明暗由 ISO、快门、光圈三者共同决定，可以用一个简化关系来理解：

曝光量 = ISO × 快门时间 ÷ f²

它们之间存在**等效关系**：当你把画面调暗一档（例如提快一档快门、减少进光）时，只要同时把光圈或 ISO 提亮一档作为补偿，最终曝光量就保持不变——这就是**等效曝光**。

正因为如此，达到同样的亮度往往有多组参数组合，区别只在副作用：快门决定运动被凝固还是拉出轨迹、光圈决定景深的深浅、ISO 牵涉噪点的多少。**先确定这张照片最想保住的那个效果，再用其余两个参数去补偿曝光**——这就是用好曝光三角的核心思路。

## 拍摄模式

理解了等效曝光，相机转盘上的 M / A / S / P 就不难理解了——它们的区别，本质上是**把哪几个参数交给相机自动决定**，对应着从全手动到全自动的不同程度。

### M — 全手动模式

ISO、快门、光圈全部由你手动设定，相机不再自动干预曝光。适合光线稳定、需要精确且可复现控制的场景，如棚拍、长曝光与视频录制。

### A / Av — 光圈优先

**你设定光圈，相机自动匹配快门速度**以保证正确曝光。这是最常用的模式之一，适合任何以景深为首要考量的拍摄：
- 人像——大光圈虚化背景、突出主体
- 风景——小光圈换取前后景皆清晰

### S / Tv — 快门优先

**你设定快门速度，相机自动匹配光圈**。当「运动如何呈现」是首要考量时使用：
- 运动、体育——高速快门凝固瞬间
- 流水、车轨——低速快门拉出动感轨迹

### P — 程序自动

相机同时决定快门与光圈的组合，你仍可拨动转盘在不改变总曝光的前提下切换不同组合（称为**程序偏移**）。适合快速抓拍，或拿不准参数时先拍到再说。

::: tip ISO 自动（Auto ISO）
在 M / A / S 模式下开启 **Auto ISO**，相机会自动调整 ISO 来兜住总曝光，同时把快门或光圈的控制权完全留给你。这是很多人从 P 模式进阶的第一步。
:::

## 对焦模式

曝光决定照片的明暗，对焦则决定画面里**哪一部分是清晰的**。常见的三种对焦模式，主要按主体是否运动来选择。

### AF-S — 单次自动对焦

半按快门完成一次对焦后即锁定，不再随主体移动。适合**静止主体**，如建筑、产品和摆拍人像。

### AF-C — 连续自动对焦

半按快门期间持续追踪主体、不断重新对焦，直到按下快门为止。适合**移动主体**，如运动员、飞鸟和街头抓拍。

> 佳能称其为 **AI SERVO**，尼康、索尼则叫 **AF-C**。

### MF — 手动对焦

手动转动对焦环，焦点完全由你掌控。适合自动对焦容易失误的场景：
- 微距摄影——景深极浅，AF 易跑焦
- 极低对比度环境——暗光、雾天
- 天文摄影——需对焦至无穷远

::: tip 对焦辅助
手动对焦时可开启**峰值对焦（Focus Peaking）**，对焦清晰的区域会高亮显示轮廓；或放大预览（通常按 MF-Assist 键）精确确认焦点。
:::

## 常见场景推荐参数

下面把前面的思路落到常见场景上，可作为起步参考；实际拍摄时再根据现场光线微调。

| 场景 | ISO | 快门 | 光圈 | 说明 |
|------|-----|------|------|------|
| 室外晴天人像 | 100 | 1/500 | f/2.8 | 背景虚化，注意过曝用 ND 滤镜 |
| 室外晴天风景 | 100 | 1/200 | f/8 | 全景清晰，画质最佳 |
| 室内无补光 | 1600–3200 | 1/60 | f/2.8 | 容忍噪点，防手抖 |
| 演出/舞台 | 3200–6400 | 1/250 | f/2.8 | 需要大光圈镜头 |
| 流水/瀑布 | 100 | 1/4–2s | f/8 | 需要三脚架 |
| 星轨 | 1600–3200 | 15–30s | f/2.8 | 三脚架，快门线/定时器 |
| 室内活动抓拍 | 800–1600 | 1/200 | f/2 | 高快门防动糊，大光圈补光 |
